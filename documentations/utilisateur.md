# Documentation utilisateur

## But

Le menu affiche la liste des jeux disponibles. L'utilisateur sélectionne un jeu, lit sa description et ses contrôles, puis le lance.

## Commandes (borne)

Correspondance historique “clavier → boutons borne” :

### Joystick J1
- Haut/Bas/Gauche/Droite : flèches directionnelles

### Joystick J2
- Haut/Bas/Gauche/Droite : `o` / `l` / `k` / `m`

### 6 boutons J1
- Ligne du bas : `f` `g` `h`
- Ligne du haut : `r` `t` `y`

### 6 boutons J2
- Ligne du bas : `q` `s` `d`
- Ligne du haut : `a` `z` `e`

## Naviguer dans le menu

- J1 Haut/Bas : changer le jeu sélectionné.
- Bouton **A** de J1 : lancer le jeu sélectionné.

## Quitter le menu

- Bouton **Z** de J1 : demande de confirmation.
- Gauche/Droite J1 : sélectionner OUI/NON.
- Bouton **A** J1 : valider.

Après avoir quitté le menu, l'installation historique lance une extinction de la machine (compte à rebours) : c'est fait par `lancerBorne.sh`.
